# mofa-extra

MoFA Extra modules including Rhai scripting support

## Installation

```toml
[dependencies]
mofa-extra = "0.1"
```

## Features

- Rhai scripting engine integration
- JSON processing utilities
- Script execution context
- Plugin scripting support
- Runtime extensibility helpers
