# mofa-macros

MoFA Macros - Procedural macros for code generation and boilerplate reduction

**Note:** This crate is intended for internal use only and is not published to crates.io.

## Features

- Procedural macros for code generation
- Boilerplate reduction macros
- Build-time code transformation
